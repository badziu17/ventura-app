# Ventura v5.0 — Collaborative Travel Planner

## New in v5
| # | Feature | Description |
|---|---------|-------------|
| 1 | **Roles** | Admin, User (logged in), Guest (view demo, plan but no save) |
| 2 | **Drag & Drop** | Reorder activities between days, with estimated durations |
| 3 | **Day overload** | Warning when >12h of activities planned in one day |
| 4 | **i18n PL/EN** | Full Polish and English, switchable anytime |
| 5 | **Travel Friends** | Add logged-in users as friends in profile; guest excluded |
| 6 | **Transport** | Bus, Metro, Taxi, Bike, Walk ratings per city with recommendations |
| 7 | **Local Customs** | AI tips (Japan trash cans, eating while walking, etc.) |
| 8 | **Car Rental** | Rentalcars.com, Discovercars, Sixt links in Book tab |
| 9 | **Travel Insurance** | World Nomads, Allianz, SafetyWing in Book tab |
| 10 | **Activity Voting** | Propose activities → group votes → approved at >50% |
| 11 | **Stay Voting** | Vote on accommodation options in Compare Stays |
| 12 | **Favorites** | Heart ❤️ places and restaurants |
| 13 | **Airport Transfer** | AI suggests best airport→hotel options per city |
| 14 | **MSZ Advisory** | Safety level + required documents per country |
| 15 | **Vacation Tracking** | Track leave request status per traveler |
| 16 | **Itinerary Builder** | Build day-by-day plan in Plan tab (same view as upcoming) |

## Deploy to Vercel
```bash
git init && git add . && git commit -m "Ventura v5"
# Push to GitHub, then import in Vercel (auto-detects Vite)
```

## Local dev
```bash
npm install && npm run dev
```
